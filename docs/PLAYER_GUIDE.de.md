## Fangen Sie an
- Öffnen Sie die App.
- Gehen Sie zu `Notifications` und melden Sie sich an.
- Browserbenachrichtigungen zulassen.

## Während des Turniers
- Überprüfen Sie `Live`, um Pools, Tabellen und Spiele anzuzeigen.
- Wenn eine Benachrichtigung eintrifft, gehen Sie zum zugewiesenen Ziel.
- Der Fortschritt hängt von der Rangfolge ab: Eliminierung, eine weitere Gruppenphase oder eine Klammer.

## Tipps
- Lassen Sie die Seite `Notifications` nach Möglichkeit geöffnet.
- Stellen Sie sicher, dass Benachrichtigungen nicht vom Browser blockiert werden.
