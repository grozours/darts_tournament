# Einem Turnier beitreten (Spieler / Admin)

Dieser Leitfaden behandelt die Registrierungsabläufe für:
- einzel
- Dublette
- Team

## Zugang
- In der App: `Doc` Link im Header.
- Nützliche Ansichten:
- `/?view=registration-players`
- `/?view=doublettes&tournamentId=<tournament-id>`
- `/?view=equipes&tournamentId=<tournament-id>`

---

## 1) Einzelturnier

### Spieler
1. Öffnen Sie die Ansicht `Registration`.
2. Gehen Sie zu **Einzelturniere**.
3. Bestätigen Sie, dass Ihre Registrierung in der Spielerliste angezeigt wird.

### Admin
1. Öffnen Sie die Ansicht `Registration`.
2. Überprüfen Sie die Teilnehmer an **Einzelturnieren**.
3. Vervollständigen Sie fehlende Registrierungen vor dem Start.

![Join single tournament](assets/screenshots/rejoindre-tournoi-simple.png)

---

## 2) Doublette-Turnier

### Spieler
1. Öffnen Sie `Doublettes` für das ausgewählte Turnier.
2. Klicken Sie in einer geöffneten Dublette auf **Beitreten**.
3. Geben Sie das Doublette-Passwort ein.
4. Bestätigen Sie, dass Ihr Name in den Mitgliedern erscheint.

### Admin
1. Öffnen Sie `Doublettes` für das Turnier.
2. Zusammensetzung prüfen (Mitglieder, Kapitän, Anmeldestatus).
3. Validieren Sie vollständige und registrierte Doubletten.

![Join doublette](assets/screenshots/rejoindre-tournoi-doublette.png)

---

## 3) Mannschaftsturnier

### Spieler
1. Öffnen Sie `Equipes` für das ausgewählte Turnier.
2. Klicken Sie in einem offenen Team auf **Beitreten**.
3. Geben Sie das Teampasswort ein.
4. Bestätigen Sie, dass Ihr Name im Dienstplan erscheint.

### Admin
1. Öffnen Sie `Equipes` für das Turnier.
2. Überprüfen Sie Dienstplan, Kapitän und Registrierungsstatus.
3. Stellen Sie komplette Teams vor dem Start fertig.

![Join team](assets/screenshots/rejoindre-tournoi-equipe.png)

---

## 4) Teamregistrierung von der Turnierkarte

### Spieler
1. Öffnen Sie die Hauptturnieransicht (`/?status=OPEN`).
2. Wenn Sie kein Team in diesem Turnier haben, klicken Sie auf **Mein Team erstellen**.
3. Sobald Ihr Team vollständig ist (4 Spieler), kehren Sie zur Turnierkarte zurück.
4. Klicken Sie auf **Registrieren**, um das Team für das Turnier zu registrieren.

### Admin
1. Überprüfen Sie, ob auf den Teamkarten je nach Spieler-/Gruppenstatus die erwarteten Aktionen angezeigt werden.
2. Stellen Sie sicher, dass unvollständige Teams nicht über die Karte registriert werden können.

![Team registration from tournament card](assets/screenshots/inscription-equipe-carte.png)

---

## 5) Verwalten Sie Ihr Team

### Spieler (Kapitän)
1. Öffnen Sie die `Equipes`-Ansicht für das Turnier (`/?view=equipes&tournamentId=<tournament-id>`).
2. Verwalten Sie Ihr Team mit den verfügbaren Aktionen: **Bearbeiten**, **Gruppe registrieren**, **Verlassen**, **Passwort**.
3. Überprüfen Sie vor der Registrierung die Mitglieder und den Gruppenstatus.

### Admin
1. Überprüfen Sie die Zusammensetzung und den Registrierungsstatus jedes Teams.
2. Bei Bedarf mit Verwaltungsmaßnahmen eingreifen.

![Manage your team](assets/screenshots/gerer-equipe-actions.png)

---

## Domain-Erinnerung
- **Single**: Einzeleintrag.
- **Doublette**: 2-Spieler-Gruppe.
- **Team**: 4-Spieler-Gruppe.
- Die Registrierung ist möglich, wenn der Spieler/die Gruppe vollständig und registriert ist.
