# Guia admin

Guia funcional para operar um torneio de ponta a ponta.

## Percurso visual
- [Walkthrough (PT)](./APP_WALKTHROUGH.pt.md)
- Referências completas: [EN](./ADMIN_GUIDE.md), [FR](./ADMIN_GUIDE.fr.md)

## Operação recomendada
- Criar e configurar o torneio.
- Rever inscritos e composição dos grupos.
- Operar fases de grupos e brackets.
- Usar a vista de alvos para iniciar, pontuar, terminar ou cancelar partidas.

## Checklist operacional
- Torneio configurado.
- Inscrições validadas.
- Fases/quadros iniciados corretamente.
- Resultados fechados de forma consistente.
