# API

Samenvatting van de REST-API en realtime-events.

## Dekking
- Endpoints voor toernooien, spelers, groepen, fasen en brackets.
- Operaties voor wedstrijden en targets.
- Authenticatie en rolgebaseerde autorisatie.
- WebSocket-events voor live-status.

## Veelgebruikte endpoints
- Service-health: `GET /health`.
