# API

Zusammenfassung der REST-API und Echtzeit-Ereignisse.

## Umfang
- Endpunkte für Turniere, Spieler, Gruppen, Phasen und Brackets.
- Match- und Target-Operationen.
- Authentifizierung und rollenbasierte Autorisierung.
- WebSocket-Events für Live-Status.

## Häufige Endpunkte
- Service-Health: `GET /health`.
