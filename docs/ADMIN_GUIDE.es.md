# Guía admin

Guía funcional para operar un torneo de principio a fin.

## Recorrido visual
- [Walkthrough visual (ES)](./APP_WALKTHROUGH.es.md)
- Referencias completas: [EN](./ADMIN_GUIDE.md), [FR](./ADMIN_GUIDE.fr.md)

## Operación recomendada
- Crear y configurar torneo.
- Revisar inscritos y composición de grupos.
- Operar fases de grupos y cuadros.
- Usar vista de dianas para lanzar, puntuar, terminar o anular partidos.

- Escenarios anónimos: [APP_WALKTHROUGH.es.md#anonymous-scenarios](./APP_WALKTHROUGH.es.md#anonymous-scenarios)

## Checklist operativa
- Torneo configurado.
- Inscripciones verificadas.
- Fases/cuadros iniciados correctamente.
- Resultados finalizados y consistentes.
