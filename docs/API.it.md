# API

Sintesi della API REST e degli eventi realtime.

## Copertura
- Endpoint per tornei, giocatori, gruppi, fasi e tabelloni.
- Operazioni su match e bersagli.
- Autenticazione e autorizzazione per ruolo.
- Eventi websocket per stato live.

## Endpoint comuni
- Salute del servizio: `GET /health`.
